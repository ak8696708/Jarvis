# Jarvis

Simple Node.js/Express project skeleton for the Jarvis assistant.

## Quick start

- Install deps (if any):

```bash
npm install
```

- Run server:

```bash
node server.js
```

## Files

- `server.js` — app entry
- `auth.js` — authentication helpers
- `index.html` — frontend
- `pakage.json` — project manifest (note: file name has a typo)
